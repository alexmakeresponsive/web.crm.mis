# TypeScript Microservice App

## Description


### Client

* **"main"**: Angular v.8 web browser app with routing


### Application Server

* **authentification**
* **"main"**
* **"msa"**


### Database

* **session** data: MongoDB v.3
* **authentification** data: MariaDb v.10
* **"main"** data: MariaDb v.10
* **"msa"** data: MariaDb v.10
